# Cricket Scoreboard

This is a simple C++ program to create a cricket scoreboard.

## Sample Input and Output

```sh
PROGRAM FOR CRICKET SCOREBOARD

Enter the number of cricket players : 4

Enter Player Info:

NAME    RUNS    BALLS
---------------------
Virat   150     120
Sachin  120     100
Dhoni   100     100
Rohit   80      50

SCOREBOARD:

NO.     NAME    RUNS    BALLS   STRIKE RATE
-------------------------------------------
1.      Rohit   80      50      160
2.      Virat   150     120     125
3.      Sachin  120     100     120
4.      Dhoni   100     100     100
```
